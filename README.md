Project Description
_____________________

This is a basic recipe website project.
It contains an index page and multiple recipe pages.

Goal
_____________________

The goal of this project is to practice HTML fundamentals, Git/GitHub workflow, and improve development skills using WSL2.

Skills Practiced
_____________________

HTML structure (headings, paragraphs, links, images)
File and folder organization in a project
Git basics (clone, add, commit, push)
GitHub repository management
Getting to know more about Images (Copyrights and attribution)
Working in WSL2 environment
